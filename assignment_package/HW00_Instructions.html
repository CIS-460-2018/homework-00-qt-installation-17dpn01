Installing Qt Creator

CIS 277 Homework 0

Due: Thursday, January 22 at 11:59 PM


1 Goal

In CIS 277, we will be using Qt in nearly all of the homework assignments. In order for you to become familiar with Qt, and to make sure that it runs properly on your computer, you will install Qt Creator and run a simple test program that we supply you.

2 Requirements

2.1 Install Qt Creator

Go to the Qt web site and download one of the installers for Qt, though we recommend the online installer. During the installation process, just make sure that you install Qt 5.4, which is the version we'll be requiring for this course and is the version that is installed in the Moore 100 labs.

If you will not be using your laptop for this course, you may use the computers in Moore 100 A, B, or C since they all have Qt 5.4 installed on them.

2.2 Test your installation

Open the Qt project included in this homework assignment package in Qt Creator. The .pro file in the Qt_GL_Test folder is the one you should open. Once you've opened the project file, you should configure it to use Qt 5.4.0. Run the program by pressing the button with the green arrow in the lower-left portion of the UI.
